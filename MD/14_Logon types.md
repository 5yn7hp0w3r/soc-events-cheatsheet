# Раздел: Logon types

|   Тип | Название                | Описание                                                                            | Типичный источник событий                            | Пример                                                                |
|------:|:------------------------|:------------------------------------------------------------------------------------|:-----------------------------------------------------|:----------------------------------------------------------------------|
|     2 | Interactive             | Прямой вход с клавиатуры на локальную машину (Ctrl+Alt+Del).                        | Локальная консоль, KVM.                              | Вход на свой рабочий компьютер в офисе.                               |
|     3 | Network                 | Сетевой доступ к ресурсам (аутентификация для доступа к общим ресурсам).            | Сетевая файловая система (SMB, NetLogon).            | Подключение к сетевой папке \server\share.                            |
|     4 | Batch                   | Пакетное задание (например, запланированная задача).                                | Планировщик заданий (Task Scheduler).                | Выполнение скрипта .bat или .cmd по расписанию.                       |
|     5 | Service                 | Вход учетной записи службы при её запуске.                                          | Менеджер управления службами.                        | Запуск службы SQL Server или планировщика заданий.                    |
|     7 | Unlock                  | Разблокировка заблокированной рабочей станции.                                      | Рабочая станция (Winlogon).                          | Возврат к компьютеру и ввод пароля после режима ожидания.             |
|     8 | NetworkCleartext        | Сетевой вход с передачей пароля в открытом виде.                                    | Основной ввод пароля по сети (не рекомендуется).     | Вход через net use с явным указанием пароля (устаревший метод).       |
|     9 | NewCredentials          | Запуск процесса с другими учетными данными (RunAs).                                 | Команда runas /netonly.                              | Запуск PowerShell от имени другого пользователя для сетевого доступа. |
|    10 | Remote Interactive      | Удаленный интерактивный вход (RDP, удаленный помощник).                             | Служба удаленных рабочих столов (Terminal Services). | Подключение к серверу через Remote Desktop.                           |
|    11 | CachedInteractive       | Интерактивный вход с использованием кэшированных учетных данных (домен недоступен). | Winlogon.exe (кэшированные данные).                  | Вход в ноутбук вне офиса, когда нет связи с контроллером домена.      |
|    12 | CachedRemoteInteractive | Удаленный интерактивный вход с использованием кэшированных учетных данных.          | RDP с кэшированными данными (домен недоступен).      | Подключение через RDP к ПК вне корпоративной сети.                    |
|    13 | CachedUnlock            | Разблокировка с использованием кэшированных учетных данных (домен недоступен).      | Winlogon.exe (кэшированные данные).                  | Разблокировка ноутбука в самолете, где нет сети.                      |
